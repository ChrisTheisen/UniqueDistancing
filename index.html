This is my workings out for this puzzle: http://www.think-maths.co.uk/uniquedistance

Generate Board: Builds an empty board of selected cols/rows.
Solve: Solves the board for the given rows/cols/goal.
STOP: Stops the Solver.
Speed: Delay in showing updates on the board (higher number is slower)

rows: number of rows on the board.
cols: number of columns on the board.
goal: number of points you are trying to find uniqe distances of.

solutions: found solutions from clicking the 'Solve' button. select one to draw it on the board.
